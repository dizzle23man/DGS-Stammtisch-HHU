Wie heißt du?

Damit andere sehen, wer kommt. Wird nur einmal gefragt und bleibt auf deinem Gerät.

App in Edge installieren

  1. Tippe auf das Menü-Symbol (…) oben rechts in Edge
  2. Wähle „Apps"
  3. Tippe auf „Diese Website als App installieren"
  4. Namen bestätigen → „Hinzufügen" tippen
  5. Das App-Icon erscheint auf deinem Homescreen ✅

App auf iPhone / iPad installieren

  1. Öffne die Website in Safari
  2. Tippe auf das Teilen-Symbol (Quadrat mit Pfeil nach oben)
  3. Scrolle und tippe auf „Zum Home-Bildschirm"
  4. Namen bestätigen → „Hinzufügen" tippen
  5. Das App-Icon erscheint auf deinem Homescreen ✅
